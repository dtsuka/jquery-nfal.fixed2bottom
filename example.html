NFAL Fixed to Bottom - jQuery Plugin

よくあるサイドメニューをスクロールに追従させる奴のウィンドウ下部にフィットさせるバージョンです。

ある程度はHTMLとCSSの設計を考える必要があるため、既存のサイトに設置するのは少々面倒かもしれません。

$('#banner').fixed2bottom();

フッターにかぶらないようにする場合

$('#banner').fixed2bottom({
	footerTarget:'#footer',
	marginBottom:20
});

options

Name デフォルト値 説明
debugMode false trueでデバッグモード
offsetTop 'auto' 対象エレメントを上から何ピクセルずらして表示するかの設定
基本的にautoのままで使用
footerTarget null フッターに被せないようにする場合は、ここにフッターを指定
例) footerTarget:'#footer'
marginBottom 0 対象エレメントの下に空けるマージン値
liquidLayout false リキッドレイアウトか否かの設定
fixedClass 'f2t_scrolling' スクロール中に追加されるクラス名（基本的には使いません）
absoluteClass 'f2t_bottomEnd' 下端までスクロールされた時に追加されるクラス名（基本的には使いません）
